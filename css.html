Cascading Style Sheets (CSS)

Designing beautiful and responsive web pages with style

What is CSS?

CSS (Cascading Style Sheets) is a language used to style and design web pages, making them visually appealing and user-friendly. It controls layout, colors, fonts, spacing, and responsiveness. While HTML provides the structure of a webpage, CSS makes it attractive.

Why is CSS Important?

  • Separates structure (HTML) from design (CSS).
  • Makes websites consistent and professional.
  • Improves readability and user experience.
  • Enables responsive design for all devices.
  • Reduces code repetition with reusable styles.

Types of CSS

  1. Inline CSS: Styles applied directly inside an HTML tag.
  2. Internal CSS: Styles written inside the <style> tag in the head section.
  3. External CSS: A separate stylesheet linked with <link>.

Example Code

/* External CSS Example */
body {
  font-family: Arial, sans-serif;
  background-color: #f8f9fa;
  margin: 0;
  padding: 20px;
}
h1 {
  color: #0d6efd;
  text-align: center;
}
p {
  color: #555;
  font-size: 16px;
}

Advanced CSS Features

Media Queries make websites responsive:

@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}

Transitions create smooth animations:

button {
  background: #ff5733;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
  transition: transform 0.3s;
}
button:hover {
  transform: scale(1.1);
}
