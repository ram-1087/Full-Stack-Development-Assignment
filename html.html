What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It provides the basic building blocks of a website, allowing developers to define elements like headings, paragraphs, links, images, and more.

Examples of HTML Elements

  • Headings (h1 - h6)
  • Paragraphs (p)
  • Links (a)
  • Images (img)
  • Lists (ul, ol, li)
  • Tables (table, tr, td)
  • Forms (form, input, textarea)
